# Crypto Exchange

Crypto Exchange – это веб-приложение для локальной криптовалютной биржи, где пользователи могут регистрироваться, проводить транзакции, обменивать криптовалюту и управлять своими счетами. Все курсы криптовалют внутри платформы определяются исключительно активностью пользователей (локальная капитализация), а не мировыми ценами.

## Функциональные требования

1. Регистрация и аутентификация
    - Регистрация пользователей с уникальным логином и паролем.
    - Вход в систему и выход.
    - Восстановление пароля через email.
    - Двухфакторная аутентификация (по желанию).

2. Управление счетами
    - Каждый пользователь имеет баланс в фиатных деньгах и криптовалютах.
    - Возможность просматривать историю транзакций.
    - Переводы средств между счетами.

3. Транзакции и курс валют
    - Обмен одной криптовалюты на другую.
    - Ввод и вывод средств (если интеграция с внешним API).
    - Комиссия 1% с каждой транзакции.
    - Курс криптовалют формируется локально и изменяется в зависимости от спроса/предложения пользователей.

4. График стоимости криптовалют
    - График изменения стоимости каждой криптовалюты.
    - Исторические данные о ценах.

5. Админ-панель
    - Управление пользователями (изменение баланса, удаление аккаунта).
    - Редактирование курса криптовалют.
    - Управление капитализацией монет.

6. API
    - REST API для интеграции с внешними сервисами.
    - Документация API.

## Технологический стек

- Backend: Python (FastAPI/Django)
- Database: PostgreSQL/MySQL
- Frontend: React/Vue.js с красивым дизайном
- Security: JWT для авторизации, шифрование паролей
- Charts: Recharts/D3.js для графиков стоимости монет
